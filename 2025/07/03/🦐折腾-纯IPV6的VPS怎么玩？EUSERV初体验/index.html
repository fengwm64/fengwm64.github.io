1. 解决网络问题

1.1 DNS64 + NAT64

1.1.1 介绍

纯 IPV6 网络连 Github 都无法访问，所以先解决基本的网站访问问题。

我们的 Cloudflare 大善人提供了 DNS64 + NAT64 的服务，该服务简单来说就是当一个域名 DNS 解析后仅有 IPV4 地址时候，CF 会利用 DNS64 服务返回一个 IPV6 地址，然后访问该 IPV6 地址，该地址是一个 NAT64 服务器，由该服务器帮助你访问 IPV4 网站，然后将结果返回给你。

DNS64 + NAT64 保证了最基本的网站访问（⚠️ 不能直接访问 IPV4 的 IP 地址，只能域名访问）
